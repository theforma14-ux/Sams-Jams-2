# Sam's Jams

Deployable Vite + React package for the Sam's Jams concept app.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- This is a polished frontend package based on the approved visual direction.
- Restaurant content is sample data for now.
- Live nationwide search, real maps, and restaurant APIs can be wired in next.
